# ⚽ Santos Pedra Mole

Projeto de site desenvolvido para um time de futebol amador da cidade de Pedra Mole (SE), com foco em prática de desenvolvimento web front-end.

Este projeto foi construído como parte dos meus estudos em HTML, CSS e JavaScript, aplicando conceitos como responsividade, layout moderno e componentes reutilizáveis.

---

## 🚀 Tecnologias utilizadas

- HTML5 (estrutura semântica)
- CSS3 (Flexbox, Box Model, responsividade)
- JavaScript (interações básicas)

---

## 💡 Funcionalidades

- Menu responsivo (desktop e mobile)
- Seção de galeria de imagens
- Cards de jogadores (artilheiros)
- Layout adaptado para diferentes telas
- Efeitos de interação no menu

---

## 📱 Responsividade

O site foi desenvolvido com foco em adaptação para:
- 📱 Mobile
- 📟 Tablet
- 💻 Desktop

---

## 🎯 Objetivo do projeto

Praticar conceitos fundamentais de desenvolvimento web e construir um projeto real para portfólio, evoluindo gradualmente para aplicações mais completas.
